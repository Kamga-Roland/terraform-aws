# terraform-create-infras

## Overview
This repository contains **Terraform configurations written in HCL** to provision and manage AWS infrastructure. It demonstrates Infrastructure as Code (IaC) practices, making deployments reproducible, scalable, and easy to maintain.

## Features
- Modular HCL code for reusability and clarity  
- AWS infrastructure provisioning (VPC, subnets, EC2, IAM, etc.)  
- Automated state management with Terraform backend support  
- Environment variables for flexible configuration  
- Best practices applied for security and scalability  

## Prerequisites
- Terraform CLI (v1.x or later)  
- AWS account with programmatic access (Access Key & Secret Key)  
- Configured AWS CLI or environment variables for authentication
